
// The robot will try to match the sentence with one of the templates below.
// If there's a match, the specified function will deal with the rest of the processing.
// The numbers indicate which words are passed to the function.
//
// Notes
//    Where there ia a *, no match is needed
//    Template parameters:
//       sentence to be matched
//       function to be called if there's a match
//       position of the word to be passed to the function starting with 1

// IS QUESTION
// parameters: type of info, subject
what is *,   is_question, class,     3  // ex: what is coffee?
what are *,  is_question, class,     3  // ex: what are raccoons?
what is a *, is_question, class,     4  // ex: what is a cow?
what * is *, is_question, attribute, 4  // ex: what color is chocolate
where is *,  is_question, location,  3  // ex: where is lucy?

// IS STATEMENT
// parameters: type of info, subject
a * is a *,  is_statement, class,     2, 5  // ex: a cat is an animal
* is a *,    is_statement, class,     1, 4  // ex: beer is a beverage
* is *,      is_statement, attribute, 1, 3  // ex: chocolate is brown

// IS CONFIRMAION QUESTION
// parameters: type of info, subject, attribute
is * *, is_confirmation_question, attribute, 2, 3  // ex: is chocolate brown
